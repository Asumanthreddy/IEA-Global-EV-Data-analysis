Electric Vehicle (EV) Data Analysis Project

Project Overview

This project analyzes global Electric Vehicle (EV) adoption trends using real-world data from the IEA Global EV Dataset (2024).
The goal is to understand EV growth, market penetration, and technology trends across different countries and years.

The analysis is performed using Power BI for interactive dashboards and CSV-based data analysis.
Project Files

IEA Global EV Data 2024.csv – Raw dataset containing global EV statistics

nova labs 11.pbix – Power BI dashboard file with visual insights
Objectives

Analyze global EV adoption trends over time

Compare EV growth across countries and regions

Understand market share of different EV technologies

Identify top-performing countries in EV adoption

Generate insights for sustainability and clean energy planning
